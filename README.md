# Day-Ahead Power Prices Analysis

This repository contains the Python notebook used to analyse European day-ahead power price data as part of an Energy consultancy coding assessment. The notebook covers data loading, preprocessing, exploratory analysis and visualisation.

## Contents

**timera_python_test.ipynb**  
Jupyter notebook including all code, analysis steps and plots.

## Objectives

Import and examine the dataset  
Calculate basic descriptive statistics  
Analyse monthly trends in day-ahead prices  
Create visualisations across multiple countries  
Demonstrate clear and efficient Python data analysis practices

## Tools & Libraries

Python  
pandas  
seaborn  
matplotlib  
numpy

## How to Use

Open the notebook in JupyterLab or VS Code to view code, outputs and visualisations.  
No other setup is required.
